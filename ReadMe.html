English

Instructions

  1. Extract to game folder.
  2. Rename OpenGL .exe to client_ogl.exe.
  3. Run Launcher.
  4. Enter key.
  5. Launch Multi Client.

VM Usage

1. Run Launcher on host PC (must be logged in).
2. Run Launcher on VM.
*Order is important.*

File Components

  • Launcher.exe - Main program
  • hwid.exe - MAC ADDR/USER etc. changer
  • watchdog.exe - connection with Launcher.exe
  • changer.dll / 32.dll - hwid.exe files
Polski

Instrukcja

  1. Wypakuj do folderu z grą.
  2. Zmień nazwę OpenGL .exe na client_ogl.exe.
  3. Uruchom Launcher.
  4. Wpisz klucz.
  5. Uruchom Multi Client.

Użycie na VM (Maszynie Wirtualnej)

1. Uruchom Launcher na głównym komputerze (musisz być zalogowany).
2. Uruchom Launcher na VM.
*Kolejność jest ważna.*

Komponenty plików

  • Launcher.exe - Główny program
  • hwid.exe - Zmieniacz MAC ADDR/USER itp.
  • watchdog.exe - Połączenie z Launcher.exe
  • changer.dll / 32.dll - Pliki powiązane z hwid.exe
Português

Instruções

  1. Extraia para a pasta do jogo.
  2. Renomeie o OpenGL .exe para client_ogl.exe.
  3. Execute o Launcher.
  4. Insira a chave.
  5. Launch Multi Client.

Uso em VM

1. Execute o Launcher no PC principal (deve estar logado).
2. Execute o Launcher na VM.
*A ordem é importante.*

Componentes de Arquivo

  • Launcher.exe - Programa principal
  • hwid.exe - Alterador de MAC ADDR/USER etc.
  • watchdog.exe - Conexão com o Launcher.exe
  • changer.dll / 32.dll - Arquivos do hwid.exe
